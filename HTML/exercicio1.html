Exercício 1 - textos em HTML

Esta é a lista de atividades para entender a utilização das tags na estruturação de um texto.

strong

Utilizado para quando você quer deixar algo em negrito.

em

Quando deseja deixar o maior ênfase em itálico.

blockquote

Texto com mais margem.

q

Quando você desejar abrir aspas dentro do texto.

abbr

É quando deseja fazer uma abreviação. Por exemplo, nos EUA é verão.

cite

Ao fazer uma citação deve-ve usar a tag cite.

dfn

Deixa o texto definido como um termo de um glossário.

ins/del/s

No você inserte um texto.. No del você deleta um texto. No s você define que esse texto não é mais correto.

sup

É um texto sobrescrito.

sub

É um texto subscrito.
